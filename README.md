# Doporučuji přečtení.
Středoškolská odborná činnost 2026
P. E. Minarčíková

Nejjednodušší a doporučený způsob spuštění je otevření projektu v prostředí Google Colab, který nevyžaduje žádnou lokální instalaci, v jiném případě je nutné stažení souboru a otevření externě.
Google Colab automaticky zajistí běhové prostředí i potřebné knihovny. Je možné spustit kód, vytvořit tabulku výsledků a vytvořit graf určených strategií.

Kód je koncipován tak, aby byl transparentní, snadno modifikovatelný a vhodný pro další rozšiřování a experimentování.
V kódu je možné volně upravovat:
- počet hráčů
- počet kol simulace
- počet vzájemných interakcí mezi hráči
- výplatní matici
